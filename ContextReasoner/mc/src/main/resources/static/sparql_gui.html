Metacognition SPARQL Console

Interfaccia web minimale per inviare query SPARQL alla REST del backend Spring Boot e visualizzare la risposta in formato tabellare o JSON.

Query

Note rapide

  • Il body inviato è JSON con campi sparql e raw.
  • La UI usa fetch() con metodo POST.
  • Se il backend non consente richieste cross-origin, va abilitato CORS nel controller Spring Boot.
  • La tabella viene mostrata solo se la risposta ha una struttura adatta.

Per servire questa pagina localmente puoi metterla tra le risorse statiche di Spring Boot, ad esempio in src/main/resources/static/.
