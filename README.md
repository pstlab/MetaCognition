# MetaCognition
MetaCognition Representation and Reasoning Component developed within the CNR-DFKI collaborative project

This repository contains the ontological model developed on top of the SOHO ontology, designed to characterize production knowledge and agents' capabilities for realizing contextual planning and execution functionalities.

The MetCognition refines and extends SOHO by introducing metareasoning skills for contextualizing the solving capabilities of planning engines available through standard frameworks, such as UP (Unified Planning Framework), with the specific needs and features of the modeled domain.
